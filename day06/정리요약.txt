자바의 메모리영역
------------------------
-메소드영역(스태틱 영역) : 클래스의 원본,static,전역변수 
-스택영역 : 함수의 호출 영역 ,지역변수
-힙 영역 : 배열,멤버변수의 복사

배열을 만드는 방법
---------------------
1) 배열을 만드는 순간 들어갈 값을 모르는경우
	int[] num = new int[10000];   //10000 은 개수! 
	int num[]    // int 뒤에 쓰는걸 더 선호한다. 
	num[0] =100;
2) 배열을 만드는 순간 들어갈 값을 이미 알고 있는 경우
	int[] age = {100,55,52,27,21}
